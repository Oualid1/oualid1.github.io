DEFINITION DE CANVAS L'élément canvas est un composant HTML qui permet d'effectuer des rendus dynamiques d'images via des scripts. Déjà implémenté dans certains navigateurs, il fait parti de HTML5.

Qu'est que c'est la balise CANVAS ?

Et quelle est son utilité ?


C'est un nouvel élément dans la partie HTML de notre navigateur (IE, Mozilla, Opera ou autres), qui peut être utilisé pour dessiner des éléments graphiques à l'aide de scripts (habituellement JavaScript).


Il permet par exemple de dessiner des graphiques, de réaliser des compositions de photographies ou des animations simples/complexes, pour présenter un produit ou une animation bien précise dans un site sans avoir besoin de logiciels bien spécifiques.

CODE qu'on peut utiliser pour créer des rectangles :


    Html                                                                Javascript

 ‹body›                                               ctx.fillStyle = 'blue';
    ‹canvas id="canvas"›                               ctx.fillRect(10, 10, 100, 50);
       ‹/body›


Avec on rajoute le Css:
        CSS canvas {border: 1px solid black;}
